# trail-map
[README.ja.md](README.ja.md)にある日本語のREADMEをご覧ください。

ウェブアプリケーションで、山と3Dテレインの情報を表示する地図を提供します。

## 機能
- 様々な山々のマーカーが表示されるマップを表示する
- 山のマーカーをクリックすると、その情報が表示され、3Dの地形ビューにアクセスできる
- ドロップダウンから山を選択して、マップの中心に移動できる

## 要件
このプロジェクトには以下のライブラリが使用されています:
- OpenLayers (v7.1.0)
- GeoLib (v3.3.3)
- Three.js

## 使用方法
1. リポジトリをクローンします: `git clone https://github.com/EiichiMiyagawa/trail-map.git`
2. `index.html` ファイルをWebブラウザで開いてアプリケーションを表示します。

## データ/API
山データと地形情報はウィキペディアと国土地理院から提供されています。
